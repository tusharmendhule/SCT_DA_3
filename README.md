# SCT_DA_3 - Interactive Dashboard using Power BI

## 📌 Objective
Create an interactive business dashboard using Power BI to visualize key sales metrics and gain insights from the Sample Superstore dataset.

## 🛠️ Tools Used
- Microsoft Power BI
- Sample Superstore Dataset

## 📊 Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Sales by Category
- Sales by Region
- Sales Trend Over Time
- Interactive Category Slicer

## 📈 Key Insights
- Technology category recorded the highest sales.
- West region contributed the highest revenue.
- Sales trends can be analyzed over time.
- Interactive filters make the dashboard user-friendly.
